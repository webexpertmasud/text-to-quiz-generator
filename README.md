# 🧠 Text to Quiz Generator

## 📌 Project Description
This is a simple web-based application that generates questions and answers from input text using FastAPI backend and JavaScript frontend.

## 🚀 Features
- Convert text into quiz questions
- Generate answers automatically
- Simple and clean UI
- Fast API response

## 🛠️ Technologies Used
- Python (FastAPI)
- HTML
- CSS
- JavaScript
